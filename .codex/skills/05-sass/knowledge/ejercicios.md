# Ejercicios

## Ejercicios localizados en el Tema 05

| Ejercicio | Trabajo solicitado | Competencia |
| --- | --- | --- |
| 1 | Crear paletas en un mapa, variables CSS y utilidades de color. | Mapas, bucles, interpolacion y tokens de color. |
| 2 | Generar tokens y utilidades desde una escala de tamanos de fuente. | Listas y generacion de clases tipograficas. |
| 3 | Generar tokens y utilidades de padding, incluidas variantes direccionales. | Escalas de espaciado y limites de generacion. |
| 4 | Repetir el enfoque anterior con margins. | Reutilizar el patron con otra propiedad de espacio. |
| 5 | Generar tokens y clases globales de familias tipograficas. | Mapas y tipografia reutilizable. |

## Que debe revisar Codex

- Que la fuente de datos sea una lista o mapa coherente.
- Que la interpolacion produzca nombres `g--` y tokens esperados.
- Que se compruebe el CSS generado, no solo el SCSS.
- Que los indices de listas Sass se traten correctamente.
- Que se detecte la inconsistencia de nombres en los tokens de color antes de decidir una solucion.
- Que no se añadan ejercicios de Material como si pertenecieran al Tema 05.

## Pendiente

Los ejemplos contienen posibles incoherencias tipograficas en salidas de padding y nombres de color. Deben contrastarse con el enunciado concreto si se implementan los ejercicios.
