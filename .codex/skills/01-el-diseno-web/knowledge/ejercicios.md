# Ejercicios del Tema 01

## Resumen de practicas

| Ejercicio | Tarea resumida | Competencia trabajada | Que debe revisar Codex |
| --- | --- | --- | --- |
| 1 | Analizar dos webs e identificar color principal y alternativo con capturas. | Reconocer jerarquia de color. | Que la justificacion se base en usos visibles y diferencie ambos colores. |
| 2 | Localizar botones de una web y clasificarlos. | Distinguir importancia y funcion. | Que cada boton tenga ambos ejes y evidencia visual. |
| 3 | Inventar ventanas con botones y describir texto, importancia y funcion. | Diseñar acciones coherentes. | Que la accion principal, la secundaria y las peligrosas no se contradigan. |
| 4 | Analizar dos webs: espacio, paleta, ancho maximo, elementos importantes, alineacion y labels. | Evaluar una interfaz de forma completa. | Que se revisen todos los criterios y se mencionen observaciones concretas. |
| 5 | Comentar el diseño de FP Valenciana a partir del articulo y la web indicados. | Critica visual y de usabilidad. | Animaciones, colores, buscador y uso de mayusculas, con referencia visible. |
| 6 | Buscar webs que cumplan o incumplan reglas, principios o mitos de usabilidad. | Argumentar sobre experiencia de usuario. | Que cada afirmacion se acompañe de un comportamiento observable. |
| 7 | Crear una pagina basada en la referencia Nemo de Wicked Templates. | Trasladar un diseño visual a una pagina. | Composicion, espacio, jerarquia, colores y fondo antes de revisar codigo. |
| 8 | Criticar la web `necesitountrastero.es` respecto al diseño. | Detectar problemas visuales reales. | Que la critica se relacione con reglas del tema, no con gustos sin justificar. |

## Criterios generales de ayuda

Cuando Codex ayude con cualquiera de estos ejercicios debe:

- pedir capturas o acceso a la referencia si la correccion depende de una web
  concreta;
- justificar observaciones con espacio, alineacion, jerarquia, color, botones,
  tipografia o usabilidad;
- distinguir entre describir un diseño y programarlo;
- no inventar el estado actual de una web externa.

## Pendientes y limites

- Las webs externas y los enlaces del tema pueden haber cambiado o dejar de
  estar disponibles; su analisis queda pendiente de comprobacion o capturas
  aportadas.
- El ejercicio 7 solicita crear una pagina, pero esta skill solo documenta sus
  criterios. No se genera codigo Angular ni CSS/SCSS en esta fase.
