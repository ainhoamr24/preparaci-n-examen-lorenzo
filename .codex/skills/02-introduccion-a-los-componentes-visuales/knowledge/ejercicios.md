# Ejercicios y practicas del Tema 02

## Alcance

En las paginas 15-23 no aparece un apartado final de ejercicios numerados. Si
se pide una lista literal de ejercicios, queda pendiente de una referencia
adicional.

El material si propone una secuencia de practicas guiadas:

| Practica guiada | Competencia trabajada | Que debe revisar Codex |
| --- | --- | --- |
| Crear la aplicacion con las opciones indicadas | Entender la base Angular del curso. | Angular CLI 20 como referencia academica, SCSS, sin SSR/SSG y sin zoneless. |
| Configurar idioma y moneda | Preparar localizacion de la app. | Que se planteen `es-ES` y `EUR`, sin afirmar que ya estan aplicados. |
| Modificar la vista inicial | Separar TypeScript y HTML del componente raiz. | Que se diferencie logica de representacion. |
| Añadir Angular Material | Incorporar la biblioteca visual utilizada en el modulo. | Que se compruebe previamente si el proyecto real ya la incluye. |
| Usar y variar un boton Material | Importar y utilizar un componente visual. | Que el componente se importe antes de usarse y se distinga su variante. |
| Añadir un icono Material | Usar otro elemento visual Material. | Que se trate como componente visual distinto del boton. |
| Personalizar botones con tokens | Centralizar cambios visuales. | Que color, forma, padding y tipografia respondan a tokens. |
| Generar y usar paletas | Entender el tema de color Material. | Que se distingan `primary`, `secondary` y `tertiary`. |

## Limites de revision

- Estas actividades son un resumen operativo del recorrido explicado, no una
  numeracion de ejercicios que el PDF no presenta.
- Cualquier implementacion queda pendiente de una peticion posterior; esta
  carpeta solo documenta.
