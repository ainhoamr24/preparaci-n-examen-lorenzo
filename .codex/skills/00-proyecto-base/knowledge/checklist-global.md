# Checklist Global

## Fuente y alcance

- [ ] La decision esta respaldada por instrucciones, una skill o el repositorio real.
- [ ] No se ha atribuido al PDF contenido no localizado.
- [ ] Se ha distinguido Angular 20 academico de Angular 21 instalado.

## Angular y estilos

- [ ] Se mantienen SCSS y componentes standalone cuando se implemente.
- [ ] No se activa SSR/SSG ni zoneless sin una nueva instruccion.
- [ ] Se respetan BEM y tokens `--mlt-sys-*`.
- [ ] Material solo se usa si procede y sus dependencias estan confirmadas.

## Interfaz

- [ ] Jerarquia, espaciado, alineacion y contraste son coherentes.
- [ ] Componentes reutilizables y paginas tienen responsabilidades separadas.
- [ ] Las dudas o decisiones adicionales estan documentadas.
