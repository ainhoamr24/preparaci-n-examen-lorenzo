# Ejercicios

## Recorrido practico localizado

El material propone:

- probar variantes de botones;
- incorporar un icono;
- personalizar componentes con Design Tokens;
- generar paletas y configurar System Tokens mediante el tema.

## Competencias

| Practica | Competencia |
| --- | --- |
| Botones e iconos | Importar y usar componentes Material. |
| Overrides de boton | Personalizar tokens de componente con SCSS. |
| Generacion de paleta | Diferenciar paletas y salida generada. |
| Tema global | Aplicar paletas y System Tokens de manera centralizada. |

## Revision de Codex

- Confirmar version y dependencia antes de implementar.
- Distinguir personalizacion de componente de tema global.
- Revisar contraste y coherencia visual.
- No anadir componentes Material no tratados como si fueran ejercicios del tema.
