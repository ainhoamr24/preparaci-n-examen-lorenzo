# Knowledge: Angular Material

## Proposito

Separar el conocimiento de Angular Material incluido en el Tema 02 para consultarlo como una skill especifica de componentes y personalizacion visual.

## Fuente verificada

Tema 02, **Introduccion a los componentes visuales**, paginas 16-23 del PDF.

## Cuando usarla

- Cuando una interfaz debe usar componentes Angular Material.
- Al razonar sobre botones, iconos, paletas, temas o tokens Material.
- Antes de personalizar estilos Material con SCSS.

## Conocimientos que aporta

- Incorporacion e importacion de componentes Material.
- Botones e iconos como ejemplos iniciales.
- Design Tokens por componente y System Tokens globales.
- `mat.theme`, paletas, tipografia y densidad.

## Relacion con Angular y diseno visual

El material del curso se orienta a Angular Material 20 y componentes standalone con imports explicitos. Los temas y tokens deben expresar jerarquia y coherencia visual, no sustituir las reglas de diseno del Tema 01.
