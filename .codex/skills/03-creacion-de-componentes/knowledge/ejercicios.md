# Ejercicios

## Practicas localizadas

| Ejercicio | Competencia |
| --- | --- |
| Componente progreso | Definir un componente y gestionar identificadores necesarios. |
| Componente panel | Modelar estructura visual reutilizable. |
| Botones por funcion e importancia | Tipar variantes y mantener coherencia visual. |
| Articulo con BEM | Identificar bloque, elementos, datos y modificadores. |
| Modificaciones condicionales del articulo | Adaptar la presentacion a datos recibidos. |

## Revision de Codex

- Verificar que el componente pertenece a UI y no a una pagina puntual.
- Revisar entradas tipadas y contenido proyectado.
- Revisar BEM y ausencia de colisiones de estilos.
- No generar librerias o dependencias mencionadas en un ejercicio sin que se pida implementacion.
