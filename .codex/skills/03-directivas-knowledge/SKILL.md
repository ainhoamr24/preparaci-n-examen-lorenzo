# Skill - Directivas Knowledge

## Objetivo

Recoger y resumir la teoría base de Angular sobre directivas para el proyecto
`PREPARACION-EXAMEN`.

## Fuente

- Documento: `teoria.pdf`
- Alcance: páginas 24 a 38
- Tema: `Tema 04 - Directivas`

## Qué cubre esta skill

- Qué son las directivas en Angular.
- Tipos de directivas.
- Directivas de componente.
- Directivas de atributo.
- Directivas estructurales.
- `@if`.
- `@switch`.
- `@for`.
- Uso de `ng-container`.

## Cómo usar esta skill en este proyecto

Antes de diseñar o modificar plantillas con directivas, consultar:

- `.codex/skills/03-directivas-knowledge/references/directivas.md`

Aplicar estas reglas:

- Elegir la directiva según el problema real: condición, selección o iteración.
- Mantener las plantillas legibles y evitar mezclar demasiada lógica.
- Usar `ng-container` cuando haga falta combinar estructuras sin añadir HTML extra.
- Preferir claridad didáctica sobre soluciones rebuscadas.

## Contexto del proyecto actual

En este repositorio la aplicación usa Angular moderno con componentes
`standalone`, por lo que la documentación de este proyecto prioriza la sintaxis
moderna de control de flujo en plantillas.
