# Ejercicios

## Ejercicio localizado

La practica del Tema 06 pide crear los componentes necesarios para una `landing-page` formada por:

- cabecera;
- lateral izquierdo;
- cuerpo;
- lateral derecho;
- pie.

## Competencias

| Trabajo | Competencia |
| --- | --- |
| Reconocer zonas de pagina | Modelar areas estructurales. |
| Distribuir cinco regiones | Seleccionar un layout adecuado, especialmente areas Grid. |
| Crear componentes necesarios | Separar composicion general y componentes internos. |
| Mantener estilos separados | Aplicar la regla de que el layout no define apariencia del contenido. |

## Revision de Codex

- Comprueba que aparecen todas las areas pedidas.
- Justifica la eleccion de Grid o Flexbox segun la estructura.
- Comprueba nomenclatura de layout compatible con BEM.
- Evita inventar paneles, navegacion, datos o interacciones.
- Si se prueba adaptacion, identifica como decision adicional cualquier breakpoint no incluido en las notas.

## Pendiente

No se localizan ejercicios adicionales de navbar, dashboards, galerias o formularios en las paginas 82-95.
