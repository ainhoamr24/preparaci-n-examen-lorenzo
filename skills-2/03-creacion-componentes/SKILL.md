# Creación de componentes

Skill de estudio para el tema **Creación de componentes** de la primera evaluación de DIW/DAW.

## Cuándo usarla

- Cuando el usuario pregunte por este tema, por sus ejercicios o por preparación de examen.
- Cuando necesite explicar conceptos, resumir apuntes, crear preguntas tipo test, hacer esquemas o resolver dudas prácticas.
- Cuando haya que generar ejemplos de HTML, CSS, SCSS o Angular relacionados con el tema.

## Flujo de trabajo

1. Lee `references/notes.md` antes de responder sobre detalles concretos del tema.
2. Si el usuario está estudiando, prioriza respuestas claras, con ejemplos cortos y pasos accionables.
3. Si pide ejercicios, usa el estilo y vocabulario de los apuntes como fuente principal.
4. Si pide código, respeta las convenciones del tema y explica solo lo necesario para que pueda reproducirlo.
5. Si una pregunta mezcla varios temas, usa esta skill para este bloque y pide o consulta la skill del otro tema cuando haga falta.
